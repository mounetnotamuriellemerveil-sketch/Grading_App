# GradingApp 
An Android application built with **Kotlin** that imports student data from an **Excel (.xlsx) file** and automatically calculates grades based on marks.This project demonstrates Android development, file processing, and data handling using Kotlin.
 Features
* Import student data from an Excel file
* Automatically calculate grades from marks
* Display student results in the app
* Simple and user-friendly interface
* Built using Android Studio and Kotlin
Technologies Used
* Kotlin
* Android Studio
* Apache POI (Excel file processing)
* Android SDK
* RecyclerView

 How to Run the Project
1. Clone the repository
2. Open the project in **Android Studio**
3. Sync Gradle dependencies
4. Run the application on an **Android Emulator or Physical Device**
 How to Use the App
1. Open the application
2. Click **Import Excel File**
3. Select a `.xlsx` file containing student data
4. The app will calculate and display the grades
